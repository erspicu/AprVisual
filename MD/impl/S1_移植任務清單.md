## Step 1 — `WireCore.Parse.cs`：`.js` 模組格式載入器 ✅

對應 `wire_defs.cpp`（`wire_defs::Load` / `loadJsonFile` / `loadExternalFiles` / `deserialize<...>` 系列）。實作：`Sim/JsModuleFormat.cs`（`JsLexer` + `JsReader`）+ `Sim/WireCore.Parse.cs`（`LoadModuleDef` + entry parsers）+ `Test/TestRunner.cs`（`--dump-module` / `--system-def-dir`）。

- ✅ **1a** JS-ish lexer（`JsModuleFormat.cs` `JsLexer`）：跳 whitespace、**逗號當空白**（結構靠 `{}[]:=` 即可，省掉所有 comma 處理）、`//` 行註解、`/* */` 區塊註解、行號追蹤；token = `{ } [ ] : =`、字串（單/雙引號）、數字（含負號 + 容忍 `.0`）、識別字
- ✅ **1b** reader（`JsReader`）：`ExpectVarHeader` / `ReadObject(onKey)` / `ReadArray(onElement)` / `BeginArray`+`AtArrayEnd`+`EndArray`（fixed-shape array 用，`EndArray` 自動跳過剩餘元素）/ `ReadString`（容忍 bare ident）/ `ReadInt` / `ReadBool` / `TryReadBool`（optional 的 IsWeak 用）/ `PeekKind` / `PeekText` / `SkipValue`
- ✅ **1c** `LoadModuleDef(dir, name)`：吃 `var module =`，`ReadObject` 按 key 分派填 `ModuleDef`（全部欄位）；遞迴載入 `modules` 子模組；**cache 於 `_loadedDefs`（keyed by name）** 讓重複型別（2× `SRAM2K`、2× `74LS368`、2× `nes-pad`）只 parse 一次（Step 2 用 `LoadedDefs` 查）
- ✅ **1d** 外部檔載入（`LoadExternalArray`）：`nodenames_files`/`transdefs_files`/`segdefs_files` → 相對於模組目錄載入 `var nodenames = {...}` / `var transdefs = [...]` / `var segdefs = [...]`，併進同一個 `ModuleDef`
- ✅ **1e** `transdef` 解析（`ReadTransDef`）：`[name, gate, c1, c2]` 後若是 `[` 跳一個陣列（bbox）、若再是 `[` 跳一個陣列（geom）、若是 `true`/`false` 讀成 `IsWeak`
- ✅ **1f** `segdef` 解析（`ReadSegDef`）：`[node, pull, layer, x1,y1,...]` — 取 `node` 與 `pull`（`'+'`/`'-'`/無），`EndArray` 跳掉 polygon 座標
- ✅ **1g** `noderef` 解析（`ReadNodeRef`）：數字 → `NodeRef(int)`；字串 → `NodeRef(string)`
- ✅ **1h** 容錯：trailing comma（逗號當空白）、`var nodenames ={`、引號/bare key、`/` `#` `~` `_` 前綴名字（出現在引號 key 裡）、未知 key（`SkipValue`）
- ✅ **驗收**：`--dump-module` 對 `74HC04`（6 trans / 6 seg `+` / 14 pin / 0 sub）、`SRAM2K`（9 trans / 1 seg / 24 pin / memory `ram(2048)`）、`2a03`（1953 named node / 27711 segdef [+:14493 −:13218] / 10946 transdef / 40 pin / 外部檔 3 個 / 內部暫存器名 `a0 x0 y0 pcl0` 都在）、`cart-mmu0`（3 conn / 1 sub `edge→nes-cart72`）、`nes-001`（33 node / 82 conn / 13 sub，遞迴載完 13 種 def 含 `4021`/`pslatch`；2C02 = 7130 node / 16806 trans / 35616 seg）全部解析無誤；`dotnet build` 0 警告 0 錯誤
- ⏳ **尚未**：`--rom` 仍丟「parse not implemented」之後的 stub（`AddInstance`/`Reset`/`StepCycle` 還沒實作）— 那是 Step 2+

## Step 2 — `WireCore.Module.cs`：模組組裝 ✅

對應 `wire_module.cpp`（`Wires::addInstance` / `setupNodes`/`setupPins`/`setupMemory`/`setupSegments`/`setupTransistors` / `addConnection` / `addNode` / `addTransistor`）+ `wire_node_resolver.cpp`。實作：`Sim/WireCore.Module.cs`（組裝邏輯 + `Node` build-time class）+ `Sim/WireCore.System.cs`（`ComposeSystem`）+ `Test/TestRunner.cs`（`--dump-system`）。

- ✅ **2a** `AddInstance(def, prefix)`：新 prefix → `AllocNodes(alignment, maxLocalId+1)`（`alignment = maxNode<100 ? 100 : 1000`）切一段；本地 id `Remap`（`vss`→`Ngnd`、`vcc`→`Npwr`、其餘 `nodeStart+id`）；`setupNodes`（每個 nodename：`AddNode(Remap(id), prefix.name)`）+ `setupPins`（加 `#<pin>` alias）+ `setupMemory`（建 `Memory`）+ `setupSegments`（pull-up 計數）+ `setupTransistors`；**已 setup 的 prefix 會重用**（只重跑 sub-module 遞迴 / connections / forceCompute）— 卡匣的 prgrom/chrrom/extraram 用 prefix `""`（→ instantiated under `"cart"`，重用 cart 實例）就靠這個
- ✅ **2b** `AddTransistor(name, gate, c1, c2, isWeak)`：`(gate==c1==c2 == EmptyNode)` 跳過；`c1==c2` 跳過；`IsPwrGnd(c1)` → swap 讓 supply 在 c2；**dedup by `(gate,c1,c2)`**（`HashSet`）；append `List<Transistor>`；登記 `Node.Gates` / `Node.C1c2s`
- ✅ **2c** segdef `'+'` → `Node.Pullups++`；`pullups:[]` 也算（`ResolveNodes` 展開後 `++`）。註：MetalNES 的 2A03/2C02 資料裡**所有** transistor 的第 7 欄都是 `false`（weak: 0），所以 `IsWeak` 目前不影響任何東西 — plumbing 留著
- ✅ **2d** `AddConnection(from, to)` = `AddTransistor("from<>to", gate:Npwr, c1:from, c2:to)`（永遠導通）；`AddConnection(fromExpr, toExpr)` 用 `ResolveNodes` 展開，one-to-many（左邊 1 個）或 one-to-one（兩邊等長）
- ✅ **2e** `ResolveNodes(expr, outIds)`：`"x|y|z"` split；`"*foo"` wildcard（掃 `_nodeByName`，前綴/後綴比對）；`"a[7:0]"` → `[a0,a1,...,a7]`（注意 index 順序對應 bit 順序，照 MetalNES，**不是**大端）；`"a[]"` → `[a0,a1,...]` 到查不到；單一名字（缺 → `Ngnd`）
- ✅ **2f** `LookupNode` / `GetNodeName`（含 `vcc`/`vss` fallback）/ `AllocNodes` / `AddNamedNode`（callback target 用）；`_nodeByName`（name→id，唯一）/ `_nameByNode`（id→name，first wins，alias 也都進 `_nodeByName` 給 wildcard 用）/ `_maxNodeId`；`ResetBuild()` 清全部 build state + 重註冊 `vcc`/`vss`；`MaxNodeIdOf`、`CombinePrefix`、`GetOrCreateNode`
- ✅ **驗收**：`--dump-system`（compose `nes-001` + `cart-mmu0` + prgrom/chrrom/extraram）→ global node array 41218 / 非空 node 15164 / pull-up 6398 / transistors 27305（含 527 connection-transistors，dedup 後）/ forceCompute 8（`ppu.spr_d[7:0]`）/ memories `u1.ram` `u4.ram` `cart.prg.rom` `cart.chr.rom` `cart.eram.ram`；`LookupNode` 對 `clk`/`res`/`cpu.clk0`/`cpu.a0`(=13737, 內部累加器!)/`cpu.ir0`/`ppu.clk0`/`u1.cs`/`u3.1/Y1`/`cart.prg.a0` 全部回合理 id；`ResolveNodes` 對 `cpu.ab[15:0]`(16)/`cpu.db[7:0]`(8)/`cpu.a[7:0]`(8)/`cart.edge.cpu_a[14:0]`(15) 正確，`*.vss`/`*.vcc` 全部 fold 到 `Ngnd`(2)/`Npwr`(1)；**整個 compose stderr 0 行**（無 connection-failed / unknown-node / missing-pin warning）；`dotnet build` 0/0

## Step 3 — `WireCore.cs Reset()`：建攤平陣列 + LUT

對應 `wire_module.cpp` `wire_compute::reset()`（~L1400-1517）。

- ⬜ **3a** alloc unmanaged 陣列：`NodeStates[NodeCount]`、`NodeInfos[NodeCount]`、`RecalcList[NodeCount]`、`RecalcListNext[NodeCount]`、`RecalcHash[NodeCount]`、`RecalcHashNext[NodeCount]`、`FlagsToState[256]`、`_groupBuf[NodeCount]`
- ⬜ **3b** `BuildFlagsToStateTable()`（已實作，呼叫它）
- ⬜ **3c** 每個 node：`NodeInfos[n].Connections = c1c2s.Count + gates.Count`；`SetFloat`（清 SetLow/SetHigh）；若 `pullups > 0` → flag `PullUp`、`NodeStates[n] = 1`；若有 callback → flag `HasCallback`
- ⬜ **3d** 建 `TransistorList`：起頭塞一個 0；每個 node：`TlistGates` = 攤平 gates 的 `(c1,c2,...)` + 結尾 0；channels 分三組 — 對普通節點 `TlistC1c2s = (gate,other,...)` + 0、far end 是 ngnd 的 `TlistC1gnd = (gate,...)` + 0、far end 是 npwr 的 `TlistC1pwr = (gate,...)` + 0（注意 gate 是 ngnd 的 channel 要跳過，照 MetalNES `setupTransistors` 那段）
- ⬜ **3e** `NodeStates[Ngnd]=0; NodeInfos[Ngnd].Flags=Gnd; NodeStates[Npwr]=1; NodeInfos[Npwr].Flags=Pwr`
- ⬜ **3f** 套用 `forceCompute` 清單 → flag `ForceCompute`
- **驗收**：`Reset()` 跑完不爆；隨機抽幾個 node 檢查 `TlistC1c2s`/`TlistGates` 內容對得上 build-time 的 gates/c1c2s

## Step 5 — `WireCore.Recalc.cs`：傳播迴圈 + step

對應 `wire_module.cpp` `recalcNodeList` / `processQueue` / `recalcNode` / `setNodeState` / `enqueueNode` / `step_cycle`（~L1519-1928, L730-773）。

- ⬜ **5a** `EnqueueNode(nn)`：`if RecalcHashNext[nn]==0 { RecalcListNext[RecalcListNextCount++]=nn; RecalcHashNext[nn]=1 }`（npwr/ngnd 跳過）
- ⬜ **5b** `SetNodeState(nn, newState)`：`if NodeStates[nn]!=newState { NodeStates[nn]=newState; 走 TlistGates: while *p: c1=*p++,c2=*p++; EnqueueNode(c1); if newState==0 && c2!=Npwr && c2!=Ngnd EnqueueNode(c2) }`
- ⬜ **5c** `RecalcNode(nn)`：`newState = ComputeNodeGroup(nn)`；`for gnn in _groupBuf[0.._groupCount): SetNodeState(gnn, newState)`；若 `_groupFlags & HasCallback` → 對 group 內有 callback 的 node `EnqueueCallback`
- ⬜ **5d** `ProcessQueue()`：`while RecalcListNextCount!=0 { iteration++; if iteration==100 Console.Error warning; swap List/ListNext + Hash/HashNext + counts; for nn in RecalcList: if RecalcHash[nn]!=0 { RecalcNode(nn); RecalcHash[nn]=0 } RecalcListCount=0 } InvokeCallbacks()`
- ⬜ **5e** `RecalcNodeList(nn)` / `RecalcNodeList(span)`：`EnqueueNode(...); ProcessQueue()`（已有外殼）
- ⬜ **5f** `SetHigh/SetLow/SetFloat(nn)`：改 flag 後 `RecalcNodeList(nn)`；`IsNodeHigh`（已有）
- ⬜ **5g** `StepCycle()`：toggle `ClockNode`；`RunHandlerChain()`；`if TraceLevel!=0 DumpTrace`；`Time++`。`Step(count)` loop（已有外殼）
- **驗收**：手刻 inverter — `SetHigh("a")` 後 `IsNodeHigh("y")==false`、`SetLow("a")` 後 `==true`；NAND、pass transistor、dynamic latch 各跑出期望真值表/phase 行為（對應 `MD/struct/04 §13.3` / `chip_tests.cpp` 的 pslatch/4021）
