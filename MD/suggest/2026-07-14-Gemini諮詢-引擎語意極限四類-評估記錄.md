## 值得日後評估的建議(依價值排序)

### 1. ⭐ Polygon 面積當電容代理(取代連接數)——「便宜大獎」候選
- 出處:CMOS 物理($C = C_ox·W·L$,gate/diffusion 電容正比面積);segdefs 多邊形就有面積資料。
- 作用:浮接平手判定改用「相連多邊形面積和」——OpenBus err4 取證時我們實測過連接數代理
  **與物理反相關**(晶片內部樞紐節點連接數多、板級走線電容大),此建議直擊那個要害。
- 成本:純載入期計算,熱路徑零成本。
- ⚠️ 代價:**黃金 checksum 必變** + 浮接判定全面重擲 → 141 顆 AC + 147 顆回歸全量重驗,
  是「大爆炸」級變更。可能從根本上消掉一批 D 類彩票,也可能翻出新偏差。
- 評估時機:掛牌跑入帳後、且有整段空檔時;先寫離線工具量「面積 vs 連接數」在既有
  爭議節點(db4、idl、#14039…)上的排序差異,再決定要不要真改。

### 2. ⭐ 正準節點排序(canonical ordering)—— D 類彩票的根治候選
- 出處:生產級模擬器的 deterministic topological sorting / canonical graph labeling(Nauty)。
- 作用:載入期按 `(距 VCC/GND 拓撲距離, 連接數, 面積)` 排序後再編 id —— u7/u8 這類
  同構實例拿到局部一致的編號 → 同一場競速解出同一結果;probe effect(圖變更重擲彩票)
  也一併削弱。
- 與既有結論的關係:呼應 Socket-Pattern 諮詢(兩階段載入 + Graph Fingerprint)。
- ⚠️ 代價:checksum 必變 + class-major renumber(R-1 win)是效能敏感區,排序準則必須與
  renumber 相容;同樣是大爆炸級。
- 附註:Gemini 建議的「evaluate-then-update 雙相更新」我們**已有**(double-buffer settle),
  它誤以為我們是連續就地更新 —— 彩票來源是浮接 tie-break 與 settle 順序,不是就地更新。

### 3. MOSSIM II 強度等級(A1 比例式打架的通用解)
- 出處:Bryant 1984,MOSSIM II conductance classes。
- 作用:從 segdefs 幾何粗算 W/L → 給電晶體 2-bit 強度 → LUT 從 256 擴到 1024/4096
  (仍在 L1)→ 比例式打架可分級解析,LAE/LXA 或可去 shim。
- ⚠️ 保留意見:LAE 的 magic 值是**晶片個體/溫度相依**($FF 是 NTSC G-rev 共識),
  離散強度分級未必能重現特定晶片的類比比值 —— 可能只是把 shim 換成另一種近似;
  且 segdefs 的 W/L 萃取品質未知。優先級低於 1、2。

### 4. Pad 佇列分層(A2 時窗重疊的通用解)—— 高度存疑
- 出處:VHDL/Verilog delta cycles / event stratification。
- 作用:pad 驅動事件進慢佇列,內部 settle 靜止後才排水 → 毛刺先在內部解掉。
- ⚠️ 保留意見:**佇列就是我們的熱路徑**(RecalcList/ProcessQueue 是多年優化的核心),
  加一層分流 = 動熱路徑,歷史上「加成本必輸」;且 A2 已由 open-bus shim 以零熱路徑
  成本解決。除非未來出現大量同族偏差,否則不值得。

## 外部驗證(對敘事有用)

- **Shim 架構被判定為業界標準做法**:等價於 Verilog-AMS 混階模擬中的 `force/release` /
  PLI/VPI callback —— 生產 EDA 界公認 switch-level 模型處理不了類比競速,行為層覆寫
  是正解;「別做通用 timing annotation 層,IRSIM 在 80 年代已證明線性 RC 追蹤會壓垮
  熱路徑」。→ 可引用進 ReportAC §3 / 論文的方法論辯護。
- **儀器 API 方向確認**:「探針永不改圖、只用旗標覆寫 + post-settle hook」= 我們的
  InstClamp 現行設計,獲獨立背書。

## Gemini 的誤讀(記錄以免日後誤信)

- A3/A4 的機理解讀不準:我們的 settle 本就把導通 pass-transistor 兩側併入同一群;
  A4 也不是 phi 重疊 shoot-through(ACLK 管線節點存在且大致工作,問題是 commit 後
  的 DMA 對 gate 免疫 + 生效延遲被 settle 壓成零)。它開的藥(two-phase settle、
  break-before-make)是我們已有或不對症的。
