## 一句話摘要

> 把 Visual2A03 / Visual2C02 這類**開關層級晶片網表**，建成可分析、可驗證、可執行的邏輯模型，先在 CPU 上跑得對，再 emit CUDA / Verilog；最有價值的並不是 GPU 本身，而是**中間那段翻譯流水線**。
